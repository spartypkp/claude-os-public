# Privacy Policy

Claude OS is an open-source personal operating system. This privacy policy applies to the optional SMS notification feature.

## SMS Messaging

SMS notifications in Claude OS are **opt-in only and disabled by default.** The feature must be explicitly configured by the user — no messages are sent without deliberate setup.

When SMS is enabled by a user:
- Messages are sent exclusively to the phone number the user configures for themselves
- The user is both the system operator and the sole message recipient
- No messages are sent to third parties
- No personal data is collected from or shared with third parties or affiliates
- Mobile information is never shared with third parties or affiliates for marketing or promotional purposes

To opt out at any time, reply **STOP** to any message. Reply **HELP** for help.
Message and data rates may apply.

## Data Collection

Claude OS runs entirely on your local machine. No user data is transmitted to external servers by this project.

All data categories exclude text messaging originator opt-in data and consent; this information will not be shared with any third parties.
