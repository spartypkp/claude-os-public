# Terms and Conditions — SMS Notifications

**Program name:** Claude OS Personal Assistant

## Program Description

Claude OS is a personal productivity system that delivers automated SMS notifications to the system owner's mobile number. Notifications include system alerts, task completion notices, calendar reminders, and status updates. The system owner is the sole recipient.

## Message Frequency

Message frequency varies based on system activity. Typically fewer than 10 messages per day.

## Rates

Message and data rates may apply.

## Opt-Out

Reply **STOP** to unsubscribe from SMS notifications at any time. You will receive a confirmation message and no further messages will be sent.

## Help
